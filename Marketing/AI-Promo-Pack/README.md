# Chit Chat AI Promo Pack

This folder contains production-ready marketing copy and AI image prompt sets for:

- Instagram launch visuals
- Intro carousel posts
- Bio/caption packs
- App Store metadata starter

## Suggested Workflow

1. Use prompts from `ai_image_prompts.json` in your preferred image model.
2. Pair outputs with captions from `instagram_captions.md`.
3. Use bios from `bio_options.md`.
4. Apply listing metadata from `appstore_metadata.md`.

## Recommended Post Sequence

1. Teaser visual
2. Product promise
3. Feature highlights
4. Corporate mode reveal
5. Call-to-action and waitlist/app link
